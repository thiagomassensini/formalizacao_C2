# Positive Box Sector Certificate Pack

Files:

- `config.json`: parameters, hashes and summary.
- `boxes.csv`: per-box sector certificate rows.
- `boxes.json`: JSON version of the per-box rows.
- `commands.sh`: reproduces the certifier run.
- `summary.md`: concise result.

This pack is external and auditable. It feeds the Lean sector scaffold conceptually, but it does not prove the analytic enclosures inside Lean.
